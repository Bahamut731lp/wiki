# Zpracování obrazu
Zpracování obrazu je odvětví, které se zaměřuje na práci s digitálním obrazem.

## Diskrétní obrazová funkce
Digitální obraz je reprezentován maticí obrazových bodů. K průchodu obrazu používáme diskrétní obrazovou funkci, která indexuje matici obrazových bodů a pro libovolnou souřadnici vrací konkrétní obrazový bod.

## Obrazové body
Obrazový bod je objekt z oboru hodnot diskrétní obrazové funkce, který obsahuje informace o jednotlivých barevných kanálech. Každý barevný kanál obsahuje __intenzitu jasu__, neboli jak "moc barva svítí".

## Barevný prostor
Barevný prostor určuje, kolik barevných kanálů má obraz a v jakém intervalu jsou jeho hodnoty jasu.

!!! example
    Pro 8bitový RGB model jsou v obraze tři barevné kanály (Red, Green, Blue) a každý kanál má rozsah od 0 do 255.
